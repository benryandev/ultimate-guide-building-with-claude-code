# Ultimate Guide: Building with Claude Code - Companion Repo

Companion repository for "The Ultimate Guide to Building with Claude Code" article.

## Purpose
This repo contains example configurations, skills, agents, rules, and documentation
for Claude Code best practices. All examples are drawn from real production projects.

## Structure
- `examples/` -- Copy-paste-ready configuration examples
- `docs/` -- Deep-dive documentation on key topics
- `.claude/` -- This directory demonstrates the practices documented here

## Conventions
- All Markdown files use standard GitHub-flavored Markdown
- Example files should be self-contained and ready to copy
- Keep examples current with Claude Code's latest features
